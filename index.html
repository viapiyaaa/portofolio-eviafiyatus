Web-based waste detection and classification system using CNN

Sentiment Analysis of Gojek App Reviews.

Classifying user reviews into positive, neutral, or negative categories. Scraping data using Google Play Scraper. Preprocessing text data, extracting features with TF-IDF, Bag of Words (BoW) and Word Embedding. The best accuracy score of LSTM with Feature Engineering Word Embedding is 99.02%

Teknologi: Python, Pandas, NumPy, Matplotlib, Seaborn, WordCloud, Sastrawi, NLTK, scikit-learn, TensorFlow, Google Play Scraper, Streamlit

Data analysis with bike sharing dataset

The project analyzed bicycle loan patterns by day and time of day. As a result, most borrowing occurs on weekdays, especially in the morning and evening, indicating routine use. Weekends and holidays were dominated by recreational use. While early morning borrowing was common, the number of bicycles borrowed was low. This finding is useful for organizing bicycle availability according to time of day and need.

Teknologi: Pandas, Matplotlib, Seaborn, Streamlit

Image Classification of Footwear

Building a deep learning model using Convolutional Neural Network (CNN) to classify footwear images into three categories: boots, shoes, and sandals. Preprocessing the data, and dividing it into training, validation, and testing sets. Converting the model to SaveModel. TensorFlow.js, and TFLite, the accuracy is 93.88%.

Teknologi: Python, TensorFlow, Keras (MobileNet, DenseNet121), OpenCV, scikit-image, scikit-learn, Streamlit

Predictive Analysis: Obesity Category Classification

Developed a model to classify obesity levels using features like age, gender, height, weight, BMI, and physical activity. Applied preprocessing, EDA, and trained models including Random Forest, KNN, and SVM. Random Forest achieved the best accuracy at 91.84%.

Teknologi: Python, scikit-learn, Pandas, Matplotlib, Seaborn

Teknologi: Python, scikit-learn, Pandas, Matplotlib, TF-IDF, Cosine Similarity
